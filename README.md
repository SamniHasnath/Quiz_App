# 🌍 World Quiz App

A modern web-based quiz application built using Node.js, Express, EJS, and PostgreSQL.

This app allows users to test their knowledge of world capitals in a clean and interactive interface.

---

## 🚀 Features

- 👤 User login & registration
- 🌍 Random country quiz questions
- ⏱️ Timer for each question
- 📊 Score tracking system
- 🏆 Leaderboard display
- 🎨 Modern SaaS-style UI
- 📱 Responsive design

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- EJS (Embedded JavaScript)
- PostgreSQL
- HTML5 & CSS3
- JavaScript
